# TheBigAdventure (En)

University Project

November 2023 - January 2024

Video game programmed in Java inspired by adventures games where you can create your own little adventure

---

## How to launch the game ?

Use java -jar thebigadventure.jar [OPTIONS] to launch the game
*the options can be found in the user manual in the "docs" folder*

## How to play ?

You can find the rules in the user manual in the folder "docs"
